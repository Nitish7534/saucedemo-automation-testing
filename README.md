# 🚀 End-to-End Automation Testing of SauceDemo using Selenium & TestNG

## 📌 Overview

This project automates the SauceDemo web application using Selenium WebDriver and TestNG.

It covers complete end-to-end scenarios including login, product selection, cart validation, and checkout process.

---

## 🧰 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

---

## 📂 Features

* Page Object Model (POM)
* Multiple test scenarios
* End-to-end workflow automation
* TestNG report generation

---

## ▶️ How to Run

1. Clone this repository
2. Open in Eclipse
3. Run testng.xml

---

## 📊 Test Reports
TestNG generates execution reports after running the tests.

Location:
test-output/index.html
